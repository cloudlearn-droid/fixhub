# fixhub
FixHub is a full-stack issue and bug tracking system inspired by Jira and Linear, built with FastAPI and React. It helps teams manage projects, track bugs, and resolve issues efficiently using Kanban workflows, secure authentication, and scalable backend design.
FixHub is an industry-grade issue tracking platform designed for modern development teams.

It enables teams to create projects, track bugs and feature requests, assign issues,
and visualize progress using a Kanban board. The backend is built with FastAPI and
PostgreSQL, focusing on clean API design, JWT-based authentication, and scalable
database architecture, while the frontend delivers a responsive and intuitive user
experience using React and Tailwind CSS.

FixHub demonstrates real-world backend engineering practices, secure access control,
and production-ready API development.
